🛠 Edge Cases & Troubleshooting
  • Scrape didn't run / didn't fire → manually trigger via Campaign Hub. Flag the broken cron to Eric.
  • Duplicate links in scrape output → manually skip them when matching. No delete button yet (on the build list).
  • Campaign has no Cobrand promotion link → create the promotion link first in Cobrand, then import the matched links.
  • New campaign added mid-week → don't wait for the next M/W/F sync. Set up the Cobrand promotion link immediately.
  • Scrape returned zero results for a campaign → log it, move on. Don't block the rest of the workflow.
  • Not sure which label a campaign belongs to → check the Notion CRM tag. That's the source of truth.
  • Stuck or unsure → ping Eric.
